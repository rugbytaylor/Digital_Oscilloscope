This folder contains the core application firmware for the ESP32 Digital Oscilloscope project.

---

## File Overview

| File | Purpose |
|------|---------|
| `main.c` | Main application entry point: initializes ADC, display, tasks, and UI |
| `waveform_display.c / .h` | Rendering waveform samples to the LCD using LVGL |
| `LUT.c` | Lookup table for ADC calibration values |
| `config_adc_raw_data.c` | ADC configuration and helper functions |
| `ota_update.c / .h` *(planned)* | Over-the-Air firmware update system |
| `version.h` *(auto-generated by CI)* | Firmware version details (major.minor.build) |

More functional modules will be added as features grow.

## Development Workflow

Firmware is built using **ESP-IDF 5+**.

To build locally:

```bash
idf.py build
idf.py flash
